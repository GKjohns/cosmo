Every fleet has unique systems and workflows. The AEGIS Public API lets you build custom integrations that connect fleet operations data with your existing infrastructure.

**API capabilities:**
- Fleet status and vessel data (read)
- Crew rosters and assignment management (read/write)
- Sensor data feeds (streaming)
- Communications dispatch (send/receive)
- Alert management (read/write/acknowledge)
- Analytics data export (read)
- Webhook subscriptions for real-time event notifications

**Authentication and security:**
API access uses the same authentication infrastructure as the main console — API keys with scope-based permissions, rate limiting, and full audit logging. Every API call is logged with the requesting identity and timestamp.

**Documentation:**
Full API reference documentation is available with interactive examples, client libraries for common languages, and a sandbox environment for testing integrations without affecting operational data.

**Rate limits:**
Standard tier: 1,000 requests/minute. Fleet tier: 10,000 requests/minute. Streaming endpoints maintain persistent connections with configurable update intervals.
