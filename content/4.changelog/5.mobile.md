Fleet command doesn't always happen from the bridge. The AEGIS Mobile Command Terminal brings essential fleet awareness to secure mobile devices — available now in beta for authorized personnel.

**Mobile capabilities:**
- Fleet status overview with vessel positions and readiness levels
- Priority comms inbox with push notifications for urgent dispatches
- Alert management with acknowledgment and escalation
- Crew status dashboard with watch rotation visibility
- Read-only tactical overlay for situational awareness
- Secure authentication with biometric and hardware token support

**What it's not:**
The mobile terminal is designed for awareness and communication, not active command. Tactical decisions, formation changes, and weapons authorization remain bridge-only functions. This is by design — command actions require the full situational awareness that only the bridge console provides.

**Security:**
All mobile communications are encrypted end-to-end with the same quantum-resistant protocols used by the bridge console. Device loss procedures include remote wipe and automatic credential revocation.
